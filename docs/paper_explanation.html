A complete walkthrough of

Prepared for the ENSAE M1 project: Control variates for a GARCH model

0   The Big Picture

We run an MCMC chain $(x^1, \ldots, x^N)$ targeting a posterior $\pi$. We want to estimate the posterior mean of some function $f$: $$\mu_f = \mathbb{E}_\pi[f(X)] = \int f(x)\,\pi(x)\,dx.$$ The standard MCMC estimator is $$\hat{\mu}_f = \frac{1}{N}\sum_{i=1}^N f(x^i).$$ This is consistent, but it has a variance that decays only like $1/N$.

Central Question of the Paper Can we replace $f$ with a cleverer function $\tilde{f}$ such that $\mathbb{E}_\pi[\tilde{f}] = \mathbb{E}_\pi[f]$ exactly, but $\operatorname{Var}_\pi(\tilde{f}) \ll \operatorname{Var}_\pi(f)$?

The answer is yes, using a technique called control variates, imported from quantum physics into statistics.

1   Control Variates — The Basic Idea

A control variate is a random variable $Z$ satisfying $\mathbb{E}_\pi[Z] = 0$. For any scalar $\beta$: $$\tilde{f}(x) = f(x) - \beta\,Z(x)$$ still satisfies $\mathbb{E}_\pi[\tilde{f}] = \mathbb{E}_\pi[f]$, but its variance is: $$\operatorname{Var}(\tilde{f}) = \operatorname{Var}(f) - 2\beta\,\operatorname{Cov}(f,Z) + \beta^2\operatorname{Var}(Z).$$ Minimising over $\beta$ gives the optimal coefficient: $$\beta^* = \frac{\operatorname{Cov}(f(X),\, Z(X))}{\operatorname{Var}(Z(X))}.$$

The variance reduction achieved is:

$$\operatorname{Var}(\tilde{f}) = \operatorname{Var}(f)\,(1 - \rho_{f,Z}^2),$$

where $\rho_{f,Z}$ is the correlation between $f$ and $Z$. The more correlated $Z$ is with $f$, the bigger the gain.

The Hard Part You need a good $Z$ — something that (1) has mean zero under $\pi$ for free, and (2) is strongly correlated with $f$. The paper solves this problem using an idea from quantum physics.

2   The Zero-Variance Principle (from Physics)

Assaraf & Caffarel (1999, 2003) introduced the following idea in quantum Monte Carlo. Introduce:

  • An operator $H$ that maps functions to functions
  • A free "trial function" $\psi$ of your choice

And define the re-normalised estimand:

$$\boxed{\tilde{f}(x) = f(x) + \frac{H\psi(x)}{\sqrt{\pi(x)}}.} \tag{2}$$

The operator $H$ is designed so that the term $\dfrac{H\psi}{\sqrt{\pi}}$ has mean zero under $\pi$ automatically — making it a valid control variate for any choice of $\psi$. The art is choosing $\psi$ to make the variance small.

Intuition In the ideal case, if you could solve $H\psi = -\sqrt{\pi}(f - \mu_f)$ exactly, then $\tilde{f}$ would be constant, giving zero variance. Since you can't solve this in general, you approximate $\psi$ with a polynomial, and get a near-zero variance estimator.

3   Choosing the Operator $H$ (Section 2 of the paper)

The paper focuses on the Schrödinger-type Hamiltonian (Equation 7):

$$H f = -\frac{1}{2}\sum_{i=1}^d \frac{\partial^2 f}{\partial x_i^2} + V(x)\,f, \tag{7}$$

where the potential is constructed from the target:

$$V(x) = \frac{1}{2\sqrt{\pi}}\,\Delta\sqrt{\pi}, \qquad \Delta = \sum_{i=1}^d \frac{\partial^2}{\partial x_i^2}.$$

4   The Key Formula — Equation (8)

With the trial function $\psi(x) = P(x)\sqrt{\pi(x)}$, where $P$ is a polynomial, the Schrödinger operator applied in (2) simplifies beautifully to:

$$\boxed{\tilde{f}(x) = f(x) - \frac{1}{2}\Delta P(x) + \nabla P(x)\cdot \mathbf{z}(x)} \tag{8}$$

where

$$\mathbf{z}(x) = -\frac{1}{2}\nabla \log\pi(x).$$

The vector $\mathbf{z}$ is the score of the target, scaled by $-\tfrac{1}{2}$. It is automatically mean-zero under $\pi$: $$\mathbb{E}_\pi[\mathbf{z}(X)] = 0.$$ No extra simulation is needed — just differentiate the log-posterior you already compute.

Case 1 — Degree-1 polynomial: $P(x) = \mathbf{a}^T x$

Here $\Delta P = 0$ (all second derivatives vanish) and $\nabla P = \mathbf{a}$, so Equation (8) becomes simply:

$$\tilde{f}(x) = f(x) + \mathbf{a}^T \mathbf{z}(x).$$

This is exactly the classical multivariate control-variate estimator, with the score components $z_1, \ldots, z_d$ as the control variates. The optimal coefficient vector is:

$$\mathbf{a}^* = -\Sigma_{zz}^{-1}\,\sigma(\mathbf{z}, f),$$

where $\Sigma_{zz} = \mathbb{E}[\mathbf{z}\mathbf{z}^T]$ and $\sigma(\mathbf{z},f) = \mathbb{E}[\mathbf{z}\,f]$ — both estimated from the MCMC samples.

Equivalent formulation (linear regression) Finding $\mathbf{a}^*$ is identical to running an OLS regression of $f(x^i)$ on the design matrix $[\mathbf{z}(x^1), \ldots, \mathbf{z}(x^N)]^T$, and extracting the slope coefficients. The ZV estimator is then the intercept of that regression.

Case 2 — Degree-2 polynomial: $P(x) = \mathbf{a}^T x + \tfrac{1}{2}x^T B x$

Now $\Delta P = \operatorname{tr}(B)$ and $\nabla P = \mathbf{a} + Bx$, so:

$$\tilde{f}(x) = f(x) - \frac{1}{2}\operatorname{tr}(B) + (\mathbf{a} + Bx)^T \mathbf{z}(x).$$

The control variate vector now contains: $z_j$ (from the linear term) and $x_k z_j$ for all pairs $j, k$ (from the quadratic term). Total dimension: $\frac{1}{2}d(d+3)$.

5   Computing $\mathbf{z}$ for the GARCH(1,1) Model

The GARCH(1,1) model is: $$r_t \mid \mathcal{F}_{t-1} \sim \mathcal{N}(0, h_t), \qquad h_t = \omega_1 + \omega_2\,r_{t-1}^2 + \omega_3\,h_{t-1},$$ with $\theta = (\omega_1, \omega_2, \omega_3)$.

The log-posterior (up to a constant) is:

$$\log\pi(\theta \mid r) = \log\pi_0(\theta) - \frac{1}{2}\sum_{t=1}^T \left[\log h_t + \frac{r_t^2}{h_t}\right],$$

where $\pi_0(\theta)$ is the truncated-normal prior.

The score vector is $\mathbf{z} = -\tfrac{1}{2}\nabla_\theta\log\pi$, so each component $z_j = -\tfrac{1}{2}\frac{\partial}{\partial\omega_j}\log\pi$ requires differentiating through the recursion $h_t$.

Differentiating through the variance recursion

Let $\dot{h}_t^{(j)} = \partial h_t / \partial\omega_j$. These satisfy the recursions:

Parameter $j$Recursion for $\dot{h}_t^{(j)}$Init
$\omega_1$ $\dot{h}_t^{(1)} = 1 + \omega_3\,\dot{h}_{t-1}^{(1)}$ $\dot{h}_0^{(1)} = \dfrac{1}{1-\omega_2-\omega_3}$
$\omega_2$ $\dot{h}_t^{(2)} = r_{t-1}^2 + \omega_3\,\dot{h}_{t-1}^{(2)}$ $\dot{h}_0^{(2)} = 0$
$\omega_3$ $\dot{h}_t^{(3)} = h_{t-1} + \omega_3\,\dot{h}_{t-1}^{(3)}$ $\dot{h}_0^{(3)} = 0$

Then the log-likelihood gradient is:

$$\frac{\partial \ell}{\partial \omega_j} = -\frac{1}{2}\sum_{t=1}^T \left(\frac{1}{h_t} - \frac{r_t^2}{h_t^2}\right)\dot{h}_t^{(j)}.$$

Adding the prior gradient (truncated-normal: $\partial\log\pi_0/\partial\omega_j = -\omega_j/\sigma_0^2$):

$$z_j(\theta) = -\frac{1}{2}\frac{\partial\log\pi}{\partial\omega_j} = \frac{\omega_j}{2\sigma_0^2} + \frac{1}{4}\sum_{t=1}^T \left(\frac{1}{h_t} - \frac{r_t^2}{h_t^2}\right)\dot{h}_t^{(j)}.$$

6   The Two-Stage Practical Algorithm

Full Procedure (degree-1 case)

1 Run MCMC. Run RWM for $N$ iterations. Store the chain $\{x^i\}_{i=1}^N$ (after burn-in) along with $\{f(x^i)\}$ and $\{\mathbf{z}(x^i)\}$ for each sample.

2 OLS regression. Regress $f(x^i)$ on $\mathbf{z}(x^i)$ using ordinary least squares: $$f(x^i) = \alpha + \hat{\mathbf{a}}^T \mathbf{z}(x^i) + \varepsilon_i.$$ The intercept $\hat{\alpha}$ is the ZV estimate of $\mu_f$. The slopes $\hat{\mathbf{a}}$ give the optimal control-variate coefficients.

3 ZV estimator. Compute the corrected values: $$\tilde{f}(x^i) = f(x^i) + \hat{\mathbf{a}}^T \mathbf{z}(x^i),$$ and report $\hat{\mu}_{\tilde{f}} = \tfrac{1}{N}\sum_i \tilde{f}(x^i) = \hat{\alpha}$.

4 Variance comparison. Compare $\hat{\sigma}^2_f = \operatorname{Var}(f(x^i))$ vs $\hat{\sigma}^2_{\tilde{f}} = \operatorname{Var}(\tilde{f}(x^i))$. The ratio is the variance reduction factor.

Note on the two-stage vs one-stage approach The paper mentions running a short "stage 1" chain to estimate $\hat{\mathbf{a}}$, then a long "stage 2" chain to compute the ZV estimate. In practice (and as the paper's footnote confirms), you can do everything on a single chain: run the chain once, then post-process it to get both $\hat{\mathbf{a}}$ and $\hat{\mu}_{\tilde{f}}$.

7   Theoretical Guarantees (Section 5)

Unbiasedness

The ZV estimator is asymptotically unbiased if the boundary term in integration by parts vanishes: $$\lim_{r\to\infty}\int_{\partial B_r} \pi\,\nabla P \cdot \mathbf{n}\,d\sigma = 0.$$ For the GARCH model with truncated-normal priors, this holds because $\pi$ decays to zero at the boundary of the parameter space.

Central Limit Theorem

Under geometric ergodicity of the chain, $\sqrt{N}(\hat{\mu}_{\tilde{f}} - \mu_f) \to \mathcal{N}(0, V(\tilde{f}))$ where $V(\tilde{f}) < V(f)$. The sufficient condition is finiteness of the Linnik functional: $$m_j = \mathbb{E}_\pi\left[\left(\frac{\partial}{\partial\omega_j}\log\pi(\theta)\right)^2\right] < \infty, \quad \forall j.$$ This is the Fisher information of a location family, and it holds for the GARCH posterior.
